# Design Anchors

## Core

- Title: `Dagon`
- Genre: isometric-style bullet heaven
- Rendering: 2D billboard sprites in a true 3D world
- Camera: fixed perspective framing
- Progression: run-based with persistent unlocks

## Theme

- sailor protagonist
- black mire opening biome
- maritime decay
- cosmic horror
- corruption and attunement

## Mechanical Identity

- dense swarms plus meaningful threats
- harpoon-based starting weapon
- brine / mire control tools
- corruption escalates both power and danger
