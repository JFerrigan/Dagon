# Asset Rules

## Primary Path

Use imported painterly sprites for:

- main characters
- enemies
- bosses
- portraits
- polished prop art

## Use Specs For

- emergency placeholders
- silhouette tests
- throwaway prototype stand-ins

## Use Manifests For

- prompt tracking
- style and provider tracking
- inbox/generated/processed handoff
- stable Unity output paths

## Output Rules

- PNG only for imported sprite assets
- transparent background
- point-filter friendly
- readable silhouette before detail
- painterly grime is good if the silhouette stays legible
